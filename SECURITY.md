## Reporting a Vulnerability

If you discover a security vulnerability in this project, please:

1. **Do NOT** open a public issue
2. Use the "Report a vulnerability" button on the Security tab
3. I will respond within 48 hours
4. You will receive a response acknowledging the issue
5. Once the fix is confirmed, a patch will be released

## Security Best Practices

- API keys and secrets should NEVER be committed to this repository
- Use environment variables or a `.env` file for sensitive data
- Keep all dependencies up to date
- Review Dependabot alerts regularly
